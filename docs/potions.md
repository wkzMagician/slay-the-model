# 药剂系统

## 概述

药剂是消耗性物品,可在战斗中使用提供各种即时效果。

## 药剂获得方式

- 战斗奖励(普通和精英战)
- 宝藏室
- 商店购买
- 事件奖励
- 休息站

## 商店药剂价格

| 稀有度 | 价格 |
|--------|------|
| **Common (普通)** | 48 - 52 金币 |
| **Uncommon (罕见)** | 72 - 78 金币 |
| **Rare (稀有)** | 95 - 105 金币 |

## 常见药剂类型

### 攻击增强类
- **Fire Potion**: 造成伤害
- **Explosive Potion**: 对所有敌人造成伤害
- **Attack Potion**: 增加攻击伤害

### 防御/恢复类
- **Block Potion**: 获得格挡
- **Strength Potion**: 获得临时力量
- **Dexterity Potion**: 获得临时敏捷
- **Focus Potion**: 获得集中(Defect职业)
- **Entrench Potion**: 获得临时格挡增益

### 效果类
- **Energy Potion**: 获得2点额外能量
- **Liquid Memory**: 抽2张卡牌

### 特殊药剂
- **Bottled Flame**: 改变药剂效果(Defect职业)
- **Fruit Juice**: 击败敌人时治疗(Defect职业)
- **Ghost in a Jar**: 避免一次死亡(每场战斗1次)

## 药剂使用

- 战斗中消耗
- 即时生效
- 战斗结束后药剂数量重置
- 携带数量有限(通常最多3瓶)

## 药剂相关遗物

### 增加药剂掉落
- **Potion Belt**: 药剂槽+1
- **Bottle**: 基础药剂数+1
- **White Beast Statue**: 药剂掉落+1
- **Sacred Bark**: 药剂掉落+1

### 商店药剂优惠
- **The Courier**: 购买的药剂被同类型药剂替换
- **Membership Card**: 药剂价格降低50%
