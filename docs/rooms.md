# 房间机制

## 概述

在《Slay the Spire》中,玩家通过地图上的不同位置进行游戏。共有 **7 种地图位置**类型,每种类型提供不同的遭遇战利和奖励。

## 房间类型

### 1. 未知位置
- **图标**: 问号标记
- **描述**: 通常包含事件,但也可能包含非精英敌人遭遇、宝藏室或商店
- **说明**: 
  - 遭遇概率受特定遗物影响（如数珠手镯、小箱子）
  - 可解析为各种房间类型
  - 事件发生在此类房间
- **影响遗物**:
  - [Juzu Bracelet](https://slay-the-spire.fandom.com/wiki/Juzu_Bracelet) - 增加事件几率
  - [Tiny Chest](https://slay-the-spire.fandom.com/wiki/Tiny_Chest) - 增加宝藏室几率

### 2. 商店
- **图标**: 商人形象
- **描述**: 使用金币购买卡牌、遗物、药水,或移除牌堆中的卡牌
- **商店内容**:
  - 5张职业卡牌(其中1张打折50%)
    - 前两张攻击卡,接着两张技能卡,最后一张能力卡
  - 2张无色卡牌(其中1张稀有)
  - 3个随机遗物(其中1个为商店专属)
  - 3瓶随机药剂
  - 卡牌移除服务(每次商店只能使用一次)

### 3. 宝藏室
- **图标**: 宝箱
- **描述**: 1个随机遗物,可能包含金币
- **说明**:
  - 可以免费打开
  - 遗物类型和数量受遗物影响
  - **影响遗物**:
    - [N'loth's Hungry Face](https://slay-the-spire.fandom.com/wiki/N%27loth%27s_Hungry_Face) - 影响遗物类型
    - [Cursed Key](https://slay-the-spire.fandom.com/wiki/Cursed_Key) - 影响遗物类型
    - [Matryoshka](https://slay-the-spire.fandom.com/wiki/Matryoshka) - 影响遗物类型

### 4. 休息站
- **图标**: 篝火
- **描述**: 休息恢复30%最大生命值,或升级1张卡牌
- **默认选项**:
  - **休息**: 恢复30%最大生命值
  - **锻造**: 升级1张选择的卡牌
- **额外选项**(需特定遗物):
  - **举重**(Girya遗物): 永久获得1点力量,每局最多3次
  - **托克**(Peace Pipe遗物): 移除1张卡牌
  - **挖掘**(Shovel遗物): 获得1个随机遗物
  - **回想**(Act 4解锁): 获得红宝石钥匙,每局最多1次

### 5. 普通敌人
- **图标**: 剑刃图标
- **描述**: 基础怪物战斗
- **奖励**: 击败后获得3张卡牌选择、金币,可能获得药剂

### 6. 精英敌人
- **图标**: 火焰图标
- **描述**: 更强的敌人战斗
- **奖励**: 击败后获得1个遗物 + 正常奖励

### 7. Boss 战斗
- **图标**: Boss头像
- **描述**: 每一幕结束时等待玩家的强大敌人
- **奖励**:
  - 3张稀有卡牌供选择
  - 3个Boss遗物供选择
  - 额外金币奖励(通常90-110金币)
  - 1瓶药剂
- **说明**: 击败Boss会将玩家生命值完全治愈到下一幕
