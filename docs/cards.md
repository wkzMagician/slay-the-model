# 卡牌系统

## 概述

卡牌是《Slay the Spire》战斗的核心元素,玩家通过构建和优化卡组来应对各种挑战。

## 卡牌类型

### 1. Attack (攻击卡)
- **边框**: 五角形
- **描述**: 直接对敌人造成伤害,可能附带次要效果
- **例子**: Strike, Bash, Cleave, Anger等
- **特点**: 
  - 有能量消耗
  - 是主要伤害手段
  - 可能有附加效果(如眩晕、格挡等)

### 2. Skill (技能卡)
- **边框**: 长方形
- **描述**: 提供防御、Buff、Debuff等多样效果
- **特点**:
  - 通常不直接造成伤害
  - 提供格挡、回血、力量、敏捷等
  - 可能施加Debuff给敌人

### 3. Power (能力卡)
- **边框**: 圆形
- **描述**: 永久升级整个战斗,一次性效果
- **特点**:
  - 每张只能使用一次
  - 提供持续增益(如力量、敏捷)
  - 可能在特定条件下触发
  - 使用后从牌堆移除(直到战斗结束)

### 4. Status (状态卡)
- **边框**: 长方形
- **描述**: 战斗中加入的负面卡牌,使牌堆臃肿
- **特点**:
  - 阻碍玩家抽到有益卡牌
  - 可能有额外负面效果
  - **战斗结束时移除**

### 5. Curse (诅咒卡)
- **边框**: 长方形
- **描述**: 永久负面卡牌,臃肿卡组
- **特点**:
  - 无法正常打出
  - 阻碍玩家抽到有益卡牌
  - 可能有额外负面效果
  - **需要专门手段移除**(如事件、遗物、商店移除等)

## 卡牌稀有度

| 稀有度 | 描述 | 非商店基础概率 | 商店概率 |
|--------|------|--------------|-----------|
| **Common (普通)** | 灰色边框 | 60% | 54% |
| **Uncommon (罕见)** | 蓝色边框 | 37% | 37% |
| **Rare (稀有)** | 金色边框 | 3% | 9% |

### 特殊稀有度

- **Basic (基础卡)**: 初始卡组中的默认卡牌,灰色边框
- **Special (特殊卡)**: 只能通过事件等特殊手段获得

## 卡牌关键词

- **Exhaust (消耗)**: 使用后从牌堆移除
- **Retain (保留)**: 回合结束时保留在手中
- **Innate (先天)**: 保证在前几手牌中出现
- **Ethereal (虚幻)**: 回合结束时如果在手中则消耗

## 各职业卡组特点

### Ironclad (铁甲)
- **红色能量**
- **基础卡**: Strike, Defend, Bash
- **特色**: 高伤害、格挡、自我牺牲换取力量

### Silent (沉默)
- **绿色能量**
- **基础卡**: Strike, Defend, Neutralize, Survivor
- **特色**: 毒、闪避、连击、匕首

### Defect (缺陷)
- **蓝色能量**
- **基础卡**: Strike, Defend, Dualcast, Zap
- **特色**: 闪电、格挡、轨道、充能

### Watcher (观者)
- **紫色能量**
- **基础卡**: Strike, Defend, Eruption, Vigilance
- **特色**: 圣言术、起舞、起落、冥想

## 无色卡牌

- 可被所有职业使用
- 不受职业限制
- 在商店中提供(左侧罕见,右侧稀有)

## 升级机制

- 卡牌可升级以增强效果
- 升级后卡牌名称显示为金色
- 升级卡通常有:
  - 更高的伤害/格挡数值
  - 更低能量消耗
  - 额外效果
